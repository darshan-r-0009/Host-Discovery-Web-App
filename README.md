# Host-Discovery-Web-App
The Host Discovery App is a Flask-based web application that scans a given subnet to identify active hosts on a network. Built as a Final Year BCA project, it combines Python’s networking capabilities with a sleek, cyberpunk-inspired frontend using Bootstrap and FontAwesome.


#🚀 Features
🔍 Scan a subnet to detect active hosts
⚡ Multithreaded scanning for faster results
🌐 Simple web interface using Flask
🎨 Modern cyberpunk themed UI
📡 Real-time network host detection
